HTML + CSS

Durante o curso, você vai aprender a estruturar um site usando essa linguagem de marcação utilizada para estruturação e apresentação de conteúdo na web. O HTML5 é uma atualização do HTML. É uma ferramenta essencial que você deve aprender a criar sites modernos. E com regras CSS3, você pode aplicar estilo aos elementos HTML. Você aprenderá o básico dessas duas linguagens, e a partir daí, vai poder abrir a mente para as mais diversas possibilidades que existe para estilizar um site.


JavaScript + jQuery

Conhecimentos sobre HTML, CSS, JavaScript e jQuery são essenciais para quem deseja ser um desenvolvedor. Além de aprender a estruturar uma página e dar modelo à ela, você será capaz de adicionar conteúdos dinâmicos, utilizando a linguagem JavaScript. Com ela você será capaz de controlar o HTML e o CSS para manipular comportamentos na página. Já o jQuery é um framework para JavaScript, foi criado para facilitar ainda mais o desenvolvimento web, escrevendo menos e fazendo muito mais coisas incríveis. Aqui você terá uma noção de como eles funcionam. Depois, você poderá usar sua imaginação.


Bootstrap

Bootstrap é o framework que vai facilitar muito sua vida. Ele tem como objetivo tornar mais fácil o desenvolvimento de interface para páginas web. Ele disponibiliza padrões para os elementos HTML mais usados, além de elementos personalizados com o uso de elementos e classes css padrões. Com ele você será capaz de criar sites com estilo profissional, e elegante, como este da página da disciplina.


Git + GitHub

Git é um sistema de controle de versão distribuído e um sistema de gerenciamento de código fonte, com ênfase em velocidade. Ele vai trazer muitos benefícios para o desenvolvimento do seu projeto ou a criação do seu site. O GitHub é uma rede social para desenvolvedores. É um site onde podemos hospedar nossos projetos. Ele utiliza o Git para controle de versão. Nele, podemos seguir projetos de nosso interesse ou mesmo dos outros desenvolvedores.


Play! Framework

Play! Framework, é um framework que possui uma alternativa limpa e eficiente para criação de aplicações web. Seu principal objetivo é facilitar ao máximo o desenvolvimento de aplicações web com Java ou Scala, visando a produtividade dos desenvolvedores. Uma das vantagens é a persistência de dados. Eu sei, você quer aprender a usá-lo. Então, venha logo! Matricule-se na disciplina, e usufura desta e de outras tecnologias!


Mais vantagens

No final do curso, o aluno deve ser capaz de:

  • Descrever os objetivos do projeto de software e seu papel no desenvolvimento de um software
  • Definir o que são padrões de projeto e nomear um conjunto de padrões
  • Criticar a necessidade e/ou oportunidade da aplicação de padrões em um contexto
  • Descrever e criticar uma arquitetura de software
  • Aplicar boas práticas e padrões de projeto na transformação de uma especificação em um projeto flexível e reutilizável
  • Produzir um software complexo utilizando boas práticas e padrões de projeto
  • Produzir um software web que utilize tecnologias atualmente populares no mercado
  • Implantar um sistema web com persistência e apresentação profissional em uma infraestrutura de computação em nuvem
